# Formula 1 (F1) Analysis #
### A basic Data Analysis on Formula 1 using the 'Formula 1 World Championship (1950 - 2023)' dataset on Kaggle ###
1. Bar plot that showcases the top Formula 1 drivers with the most Grand Prix wins.
2. Bar plot that showcases the top 10 Formula 1 drivers with the most Grand Prix wins.
3. Bar plot that showcases the top Formula 1 constructors with the most Grand Prix wins.
4. Bar plot that showcases the top 10 Formula 1 constructors with the most Grand Prix wins.
5. Linear regression analysis to explore the relationship between a Formula 1 driver's starting position on the grid and their finishing place in the race.
6. Analysis of race data focusing on the average fastest lap speed by Grand Prix 2004 onwards using Facetgrid.
